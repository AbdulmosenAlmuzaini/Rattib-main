# خطة التنفيذ التقنية - منصة «رتّب – مكتب المعقّب الذكي»

تقدم هذه الوثيقة خريطة الطريق الفنية والتنفيذية لبناء نظام رتّب، متضمنةً التقنيات المستخدمة، الهيكل البرمجي المقترح، ومراحل العمل التفصيلية لضمان تلبية كافة متطلبات العمل والأمان.

---

## 1. البنية التقنية المعتمدة (Technology Stack)

### أ. الواجهة الأمامية (Frontend)
*   **React 19** بالاعتماد على **Vite** للتطوير والتحزيم السريع.
*   **Tailwind CSS v4** للتنسيق البصري وبناء واجهة حديثة بالكامل تدعم التوجيه من اليمين إلى اليسار (RTL).
*   **Motion** (عبر `motion/react`) لإضافة تأثيرات حركية خفيفة واحترافية للانتقال بين القوائم والصفحات ولوحة Kanban.
*   **Lucide React** لمجموعة أيقونات الواجهة الموحدة.
*   **Recharts / D3** لبناء لوحة تقارير وتحليلات مالية ومبيعات تفاعلية.

### ب. الواجهة الخلفية (Backend)
*   **Express.js** كخادم ويب سريع ومرن لتوفير واجهات برمجة التطبيقات (RESTful API) تحت المسار `/api/*`.
*   **TypeScript** كلغة برمجة أساسية في العميل والخادم لضمان أمان الأنواع وسرعة اكتشاف الأخطاء.
*   **tsx & esbuild** لإدارة تشغيل الخادم في بيئة التطوير والإنتاج بطريقة سلسة وسريعة.

### ج. قاعدة البيانات وإدارة البيانات (Database & Storage)
*   **PostgreSQL** كقاعدة بيانات علائقية متينة تدعم العلاقات المعقدة والعمليات المالية والبحث بكفاءة.
*   **Drizzle ORM** كأداة حديثة لإدارة الجداول والاستعلامات بطريقة آمنة ومتوافقة مع TypeScript.
*   **Firebase Authentication** كحل متين ومستقر ومجاني للمصادقة وتأمين الجلسات وإدارة تسجيل دخول المستخدمين والموظفين بشكل كامل.

## 2. الهيكل البرمجي المقترح للمجلدات (Project Directory Structure)

سيتم تنظيم مجلدات المشروع بطريقة تضمن الفصل التام بين منطق الخادم (Backend) ومنطق الواجهة (Frontend)، وتسهيل الصيانة:

```text
/
├── .env.example                # نموذج الإعدادات البيئية للمشروع
├── .gitignore                  # الملفات المستثناة من التتبع
├── index.html                  # ملف الواجهة الرئيسي
├── metadata.json               # معلومات ووظائف التطبيق
├── package.json                # الحزم البرمجية والاعتماديات
├── tsconfig.json               # إعدادات لغة TypeScript
├── vite.config.ts              # إعدادات محزم Vite
├── drizzle/                    # مجلد الهجرات المتولدة من Drizzle Kit
├── src/                        # المجلد البرمجي الرئيسي
│   ├── main.tsx                # نقطة الانطلاق للواجهة الأمامية
│   ├── index.css               # الأنماط والتنسيقات العالمية (Tailwind)
│   ├── App.tsx                 # المكون الجذري للواجهة والموجه الرئيسي
│   ├── types.ts                # أنواع TypeScript المشتركة وجداول البيانات
│   ├── server.ts               # خادم Express الرئيسي ونقاط الـ API
│   ├── db/                     # مجلد الاتصال بقاعدة البيانات
│   │   ├── index.ts            # تهيئة اتصال PostgreSQL و Pool
│   │   ├── schema.ts           # تعريف جداول قاعدة البيانات بالكامل (Drizzle Schema)
│   │   └── drizzle.config.ts   # إعدادات أداة Drizzle Kit للمهجرين
│   ├── middleware/             # البرمجيات الوسيطة للخادم
│   │   └── auth.ts             # التحقق من مصادقة Firebase وحقوق الوصول
│   ├── components/             # المكونات البرمجية القابلة لإعادة الاستخدام
│   │   ├── ui/                 # عناصر الواجهة الأساسية (أزرار، حقول إدخال، جداول)
│   │   ├── Dashboard.tsx       # لوحة التحكم والإحصائيات وملخص اليوم
│   │   ├── Clients.tsx         # إدارة العملاء وملفاتهم الموحدة
│   │   ├── Transactions.tsx    # إدارة المعاملات (العرض الجدولي وكانبان والتقويم)
│   │   ├── Templates.tsx       # قوالب الخدمات والمعاملات الافتراضية
│   │   ├── Documents.tsx       # مركز إدارة المستندات وتتبع تواريخ الصلاحية
│   │   ├── Tasks.tsx           # المهام والتقويم والتذكيرات
│   │   ├── Accounts.tsx        # النظام المالي وتسجيل الدفعات والمصاريف
│   │   ├── Reports.tsx         # التقارير والرسوم البيانية التفاعلية
│   │   ├── Team.tsx            # إدارة الموظفين والصلاحيات (RBAC)
│   │   ├── ActivityLog.tsx     # سجل الأنشطة والتدقيق الأمني
│   │   └── Sidebar.tsx         # القائمة الجانبية للتنقل
│   └── lib/                    # المساعدات والمكتبات المشتركة
│       ├── firebase.ts         # تهيئة Firebase Client SDK
│       ├── firebase-admin.ts   # تهيئة Firebase Admin SDK للخادم
│       └── utils.ts            # دوال مساعدة (تنسيق المبالغ، التواريخ)
```
